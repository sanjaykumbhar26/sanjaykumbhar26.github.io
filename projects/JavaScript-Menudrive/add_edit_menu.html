You can create a menu dedicated to daily specials or special offers and then restrict the number of different items allowed from the menu per order.


For example, if you set the number of different items to two, then only two different items are allowed (e.g., pizza and burger, not pizza, burger, and hoagie). Please note that coupons cannot be used with items marked as special items.
